Hobo это простой легко вессовой движок учебный проект для изучения OpenGL с использаванием библиотек
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
основные библиотки которые нужны для роботы:
  GLFW;
  GLAD;
  glm;
  stb_image.h;
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Потрачено неделю времени и допиливание около 2-3 недель скорее изучения физики и прикольных вещей
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
ЧТО ЖЕ УМЕЕТ HOBO?
  компиляция шейдеров и последуйщия загруска в .bin файлы,
  встроенная подержка ECS и не много DOD (GroupShader - это групирования для уменьшения смены стейтов что в графике больно, + он с самого начало instanced имеет),
  не много пошалил и зачем то добавил математику матрици кроме 4 и вектора 4 просто бонус,
  абстракции над VAO VBO EVO - Mesh,
  attrib - MeshFilters,
  курсор - упровления курсором там выключения включения ограничение смена позиции получения позиции и дельту,
  встроенный namespace Object готовый есть 2 объекта это plane, cube.,
  namespace Render для упровления игровым циклом.
Если что-то не вошло то значет я забыл но в целом этого я считаю достаточно.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
